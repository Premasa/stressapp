# Serenity - Employee Stress Relief App

A comprehensive wellness companion designed to help employees manage stress and maintain mental well-being in the workplace.

## Features

- **Guided Breathing Exercises**: Multiple breathing patterns including 4-7-8, Box Breathing, and Calm Breathing
- **Ambient Sounds**: Immersive audio experiences with rain, ocean waves, birds, fireplace, night crickets, gentle wind, and mountain streams
- **Stress Relief Games**: Bubble pop, color matching, and zen drawing activities
- **Mood Tracking**: Daily mood logging with analytics and history
- **Inspirational Quotes**: Daily wisdom and motivation
- **Break Reminders**: Customizable reminders to take regular breaks

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (ES6+)
- Chart.js for mood analytics
- Web Audio API for sound generation

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start using the app!

## Browser Support

Works in all modern browsers with JavaScript enabled.
